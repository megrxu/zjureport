\section{实验目的和要求}
  系统差分方程和传输函数是线性系统的重要概念，通过分析系统差分方程和传输函数的特性，编程查看系统零极点分布，加深对线性系统的了解。
\section{实验内容和步骤}

  \subsection{实验内容}

    给出如下差分方程：
    $$y(n) - (0.5+a)\times y(n-1) + 0.5ay(n-2) = x(n)$$
    \begin{enumerate}
      \item 求解系统传输函数表达式。
      \item 当a取0.8，0.9，1.0，1.1时，画出零极点分布图。
      \item 根据（2）中a的取值，分别画出幅频响应函数。
    \end{enumerate}

  \subsection{实验步骤}
    \begin{enumerate}
      \item 编写程序，求解零极点
      \item 画出图形。
      \item 观察结果。
    \end{enumerate}
